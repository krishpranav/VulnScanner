# VulnScanner
This tool will scan a website for vulnerabilities 

Steps to run this tool:

-  git clone https://github.com/krishpranav/VulnScanner
-  cd VulnScanner
- sudo chmod 777 *
- python vulnscan.py    or   sudo ./vulnscan.py
